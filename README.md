## ⌁ About

<br>

I work on backend systems and platform reliability — designing services in Java and Spring Boot, building data pipelines with Kafka, and managing observability with Grafana and Prometheus. I care about systems that are fast, predictable, and easy to debug when something goes wrong.

Right now I'm learning **Rust**, mostly by rewriting performance-critical pieces of existing systems and seeing what changes. I'm also exploring **ClickHouse** as an alternative to Elasticsearch for large-scale analytical workloads.

I write occasionally about backend engineering, distributed systems, and things learned the hard way in production → **[Read on Medium](https://medium.com/@adityaramachandran25)**
